# sz-sdk-go-mock errors

## Error prefixes

Error identifiers are in the format `senzing-PPPPnnnn` where:

`P` is a prefix used to identify the package.
`n` is a location within the package.

Prefixes:

1. `6031` - szconfig
1. `6032` - szconfigmanager
1. `6033` - szdiagnostic
1. `6034` - szengine
1. `6035` - szhasher
1. `6036` - szproduct
1. `6037` - szssadm
